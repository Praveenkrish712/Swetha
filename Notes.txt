CSS - Cascading Style sheet

1) Types of CSS
==> inline CSS
==> internal CSS
==> External CSS
inline > internal > External

2) CSS Syntax
selector{
    property: value;
}

3)Types of selector
==> Elements selector - (h1) (h2) (p)
==> class selector - (.) (no of times) - mutiple times we can use the same class name
==> id selector - (#) (Unique) - only one time we can use the same id name
==> Universal selector - (*) - to be apply with entire site
==> Group selector - class or id name containes different tags, we want to change same Style means we go with Group selector

7) flex (display) - one dimensional layout system....
i) using display property
ii) flex direction - row, row-reverse, column, column-reverse
iii) justify-content (X) - start, center, end, space-between, space-around, space-evenly
iv) align-item (Y) (set the height) - start, center, end
v) flex-wrap - no-wrap, wrap, wrap-reverse
vi) gap - to be assign with values (px,%,rem,m,cm,vh,vw)
vii) Align-content (Y) (set the height) (flex-wrap is enable then only it will work) - start, center, end, space-between, space-around, space-evenly
viii) flex (on item) - using *flex:1*
viv) order - change the visual order of the item
