HTML - Hyper text markup language( <html> </html> )

* it is not a programming language
* it is a markup language
* to create a webpage
* every site start with HTML

1) Heading - h1 to h6

2) paragraph - p

3) hr

4) br

5) Text formatting - i, u, b or strong, sub, sup and mark

6) Anchor tag - a (linking)

7) Image tag - img

8) list - ordered list and unordered list

9) Tables - tr, th and td

10) Forms - placeholder, input, required, label and button

11) Media Elements - video and audio

12) Sematic tags - Header, footer, section, nav and main


CSS - Cascading Style sheet

1) Types of CSS
==> inline CSS
==> internal CSS
==> External CSS
inline > internal > External

2) CSS Syntax
selector{
    property: value;
}

3)Types of selector
==> Elements selector - (h1) (h2) (p)
==> class selector - (.) (no of times) - mutiple times we can use the same class name
==> id selector - (#) (Unique) - only one time we can use the same id name
==> Universal selector - (*) - to be apply with entire site
==> Group selector - class or id name containes different tags, we want to change same Style means we go with Group selector
